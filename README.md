# InkLantern

配信活動の中で出会った人を、忘れないための小さな手帖です。

出会った日、名前、性別、年齢、Xアカウント、話した内容、好きな音楽、最後に会いに来てくれた日を記録できます。最後に会ってからの経過日数は自動計算され、時間が経つほどバッジの色が暖色に変わっていくので、「そろそろ声をかけたい人」に自然と気づけるようになっています。

## 特徴

- インストール不要。`index.html` をブラウザで開くだけで動作します
- サーバー・アカウント登録は一切不要
- データはすべて自分のブラウザ内（localStorage）に保存され、外部には一切送信されません
- 1ファイル完結なので、USBメモリやDiscordでの共有もそのまま可能です

## 使い方

1. このリポジトリをダウンロード（またはクローン）する
2. `index.html` をダブルクリックしてブラウザで開く
3. 「＋ 新しく記録する」から記録を追加していく

## 自分のサイトとして公開する場合

`index.html` と `icon.svg` を、GitHub PagesやNetlifyなど好きな静的サイトホスティングにそのまま置くだけで公開できます。ビルドツールや依存パッケージは一切不要です。

## データについて

すべてのデータはブラウザのlocalStorageに保存されます。そのため：

- 端末やブラウザを変えると、記録は引き継がれません
- 複数人で同じURLを開いても、記録はそれぞれの端末ごとに独立して保存されます（データが混ざる心配はありません）
